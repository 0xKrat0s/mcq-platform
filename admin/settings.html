About

MCQ Exam Platform (Static Version)

A simple and powerful platform for conducting MCQ-based examinations. This is the static version that runs entirely in the browser using localStorage.

Features:

  • Create and manage multiple exams
  • Flexible result visibility modes
  • Timer and auto-submit functionality
  • Detailed analytics and exports
  • Prevent duplicate attempts
  • No server required - runs on GitHub Pages

Note: Data is stored in your browser's localStorage. Clearing browser data will erase all exams, questions, and results.
